# Console Task Manager

## 📌 Project Description
Console application for managing tasks (To-Do List).

## 👥 Team Members
- Hris – UI, menu, colors, validation
- Tsetso – task logic, file storage, sorting

## ⚙️ Functionalities
- Add new task
- Show all tasks
- Mark task as completed
- Delete task
- Sorting by name (A–Z)
- Sorting by status
- File persistence (tasks.txt)
- Colored console interface

## 💾 Technologies
- C#
- .NET 8
- GitHub (branches, pull requests)
